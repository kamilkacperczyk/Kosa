# Instrukcje dla asystentow AI - BeSafeFish

Ten folder zawiera instrukcje projektowe w formacie neutralnym (nie zaleznym od Claude).
Mozna je uzyc do konfiguracji dowolnego asystenta AI: Gemini, ChatGPT, OpenCode, Copilot, Cursor itp.

## Struktura

- `project-instructions.md` - zasady pracy w tym projekcie (skopiuj do swojego AI)
- `global-instructions.md` - globalne zasady uzytkownika (jezyk, commity, bezpieczenstwo)
- `setup-guide.md` - jak skonfigurowac konkretne narzedzia AI na podstawie tych instrukcji

## Jak uzywac

Przeczytaj `setup-guide.md` i postepuj wedlug instrukcji dla swojego narzedzia.
